# Gmail Agent - Manuel d'utilisation

## Introduction
L'agent Gmail est un outil puissant qui permet d'interagir avec Gmail via une interface Python. Il utilise l'API Gmail et l'API Gemini pour offrir des fonctionnalités avancées de gestion des emails.
